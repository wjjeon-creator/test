# 여행 준비물 체크리스트 🧳

빌드 도구 없이 `index.html` 한 파일로 동작하는 여행 준비물 체크리스트 웹앱입니다.

## 기능
- 템플릿(해외여행 / 국내여행 / 캠핑 / 출장) 한 번 클릭으로 불러오기
- 카테고리별 그룹 표시 (서류 / 전자기기 / 의류 / 세면도구 / 기타)
- 항목 추가 · 체크(취소선) · 삭제
- 항목 클릭 시 인라인 수정
- 진행률 표시(진행 바 + `N개 중 M개 완료`)
- 다크모드 토글
- 브라우저 localStorage 자동 저장

## 실행
`index.html`을 브라우저로 열기만 하면 됩니다. 별도 설치나 빌드가 필요 없습니다.

## 배포
정적 파일이므로 Vercel에서 별도 설정 없이 그대로 배포됩니다.
